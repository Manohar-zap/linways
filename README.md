# Feedback Form Automation (AutoHotkey v2)

A lightweight desktop automation script built using AutoHotkey v2 to speed up repetitive rating form submissions.

## Features
- Toggle-based start/stop
- Randomized rating selection
- Adjustable delay timing
- Keyboard-based navigation (A/N + Tab)

## Requirements
- Windows OS
- AutoHotkey v2

## Usage
1. Install AutoHotkey v2.
2. Run the .ahk script.
3. Click inside the first dropdown field.
4. Press F1 to start automation.
5. Press ESC to stop.

## Note
This tool is intended for learning and productivity enhancement purposes only.
